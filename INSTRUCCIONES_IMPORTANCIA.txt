ACTUALIZACIÓN: IMPORTANCIA ESTRATÉGICA

1. Abre el repositorio Gibraltar-Watch en GitHub.
2. Code -> Add file -> Upload files.
3. Descomprime el ZIP y sube todos los archivos de su interior.
4. Confirma que GitHub reemplaza index.html, en.html, styles.css y el resto de páginas HTML.
5. Pulsa Commit changes.
6. No es necesario ejecutar GitHub Actions: esta actualización es de contenido y diseño.
7. Espera 1-2 minutos y recarga la web con Ctrl + F5.
