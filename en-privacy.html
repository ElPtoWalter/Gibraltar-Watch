Privacy

The site works without accounts, forms or first-party analytics. External services activate only when needed.

Data we do not collect

Gibraltar Watch does not request names, email addresses or precise location and does not create user profiles. This version installs no advertising cookies or first-party analytics.

External services

Loading the AIS radar connects your browser to MarineTraffic. The seismic map uses OpenStreetMap/Carto tiles and Leaflet. These providers may receive standard technical data such as IP address, browser and referrer.

GitHub Pages

The site is hosted on GitHub Pages. GitHub may record technical access information under its own policy.

Scientific data

JSON files contain only public earthquake and source data, not personal data.

Future advertising

Before advertising or analytics are added, an appropriate consent system will be implemented for European Economic Area visitors.
