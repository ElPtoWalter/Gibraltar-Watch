TRANSPARENCIA

Privacidad

La web funciona sin cuentas, formularios ni analítica propia. Los servicios externos solo se activan cuando son necesarios.

Datos que no recopilamos

Gibraltar Watch no solicita nombre, correo electrónico, ubicación precisa ni crea perfiles de usuario. No instala cookies publicitarias ni herramientas de analítica propias en esta versión.

Servicios externos

Al cargar el radar AIS, el navegador conecta con MarineTraffic. El mapa sísmico utiliza teselas de OpenStreetMap/Carto y Leaflet. Esos proveedores pueden recibir datos técnicos habituales, como dirección IP, navegador y página de referencia.

GitHub Pages

El sitio se aloja en GitHub Pages. GitHub puede registrar información técnica de acceso conforme a su propia política.

Datos científicos

Los archivos JSON contienen exclusivamente datos públicos sobre terremotos y fuentes, no datos personales.

Publicidad futura

Antes de incorporar anuncios o analítica se añadirá un sistema de consentimiento adecuado para visitantes del Espacio Económico Europeo.

Proyecto divulgativo independiente. Los datos sísmicos automáticos no sustituyen la información de protección civil ni la navegación profesional.
